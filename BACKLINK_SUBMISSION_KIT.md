Long description:

```text
CometPDF is a free Windows desktop app for local PDF conversion. It helps users convert Word, Excel, PowerPoint, images, Outlook emails, text files, data files, and existing PDFs without uploading documents to an online service. CometPDF is designed for private documents, office workflows, email archiving, batch conversion, and quick PDF delivery.

The app can also export PDFs to PNG, JPG, TXT, Word DOCX, and PowerPoint PPTX. PDF to Word and PDF to PowerPoint are visual exports, preserving page appearance rather than rebuilding every object as fully editable content.

CometPDF offers both an installer and a Portable ZIP. The installer can add Desktop and Start Menu shortcuts, uninstall support, and optional right-click conversion from Windows Explorer. The Portable ZIP runs without installation.
```
