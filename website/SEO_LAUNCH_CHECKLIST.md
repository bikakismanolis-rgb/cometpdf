# CometPDF SEO Launch Checklist

The current production domain is `https://cometpdf.com`. If the domain changes later, update:

- `website/index.html`
- all SEO page canonical URLs
- `website/robots.txt`
- `website/sitemap.xml`

## Publish

1. Upload `website/` or `release/CometPDF-website.zip` to Netlify, Cloudflare Pages, Vercel, or another static host.
2. Connect the final domain.
3. Confirm these URLs work:
   - `/`
   - `/sitemap.xml`
   - `/robots.txt`
   - `/offline-pdf-converter/`
   - `/word-to-pdf/`
   - `/excel-to-pdf/`
   - `/powerpoint-to-pdf/`
   - `/msg-to-pdf/`
   - `/image-to-pdf/`

## Google

1. Add the domain to Google Search Console.
2. Submit `https://cometpdf.com/sitemap.xml`.
3. Use URL Inspection on the homepage and request indexing.

## Early Backlinks

- Add the site link to a GitHub repository README.
- Share the first release on LinkedIn.
- Share carefully in relevant Windows/productivity communities.
- Ask first users to link to the site if they find it useful.
