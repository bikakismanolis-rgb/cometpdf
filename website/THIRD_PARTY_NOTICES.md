# CometPDF - Third Party Notices

CometPDF uses the following third-party open-source components:

- Pillow: image handling.
- ReportLab: PDF generation.
- pypdf: PDF merging.
- openpyxl: Excel fallback parsing.
- tkinterdnd2: optional drag-and-drop support.
- PyInstaller: Windows executable packaging.

Microsoft Office, Outlook, LibreOffice, and Windows are not bundled with CometPDF. The app only uses them when they are already installed on the user's computer.

This file is informational and should be reviewed before public or commercial distribution.
