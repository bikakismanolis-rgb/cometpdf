FAQ

Which image formats are supported?

CometPDF supports JPG, JPEG, PNG, WEBP, BMP, TIFF, and multi-page TIFF files. These cover common camera photos, screenshots, scanned pages, and web image formats.

Can I merge multiple images into one PDF?

Yes. Enable the merge option after conversion to combine all image PDFs into a single file. This is useful for creating multi-page documents from separate image files or scans.

Does it upload my images?

No. Images are converted locally on your Windows computer. CometPDF does not connect to the internet during conversion.

Can I control the page size and orientation?

CometPDF fits each image to a standard page for simple, predictable output. Output folder and file naming can be configured in the app.

Does it work with scanned documents?

Yes. TIFF and multi-page TIFF files from scanners are supported. For searchable text in scanned PDFs, OCR support is planned for a future Pro release.
