# CometPDF Link Building Plan

The goal is not spam. The goal is to put CometPDF where people genuinely look for private/offline PDF conversion tools.

## First Links To Create

- GitHub repository with README, screenshots, releases, and link to the website.
- Product Hunt launch page.
- LinkedIn post explaining the offline/no-upload angle.
- Reddit or forum posts only where self-promotion is allowed and useful.
- Personal/company website blog post: "Why we built an offline PDF converter for Windows."

## Useful Content That Can Earn Links

- "Why offline PDF conversion matters for sensitive documents."
- "How to convert Outlook MSG emails to PDF without uploading them."
- "Online PDF converters vs offline PDF converters."
- "How to batch convert Office files to PDF on Windows."

## Outreach Targets

- Small business productivity blogs.
- Legal/accounting/office workflow newsletters.
- Windows utility directories.
- Privacy-focused software lists.
- Greek small business and accounting communities for the first local test wave.

## Rules

- Do not buy spammy links.
- Do not mass-post the same message.
- Do not describe CometPDF as an online converter.
- Always say it is a Windows desktop app that converts files locally.
