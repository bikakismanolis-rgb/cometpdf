Installer Scan

The CometPDF 1.1.1 installer was rebuilt to hide helper command windows during conversion. A new VirusTotal scan should be published for this exact hash before broad public promotion. The previous 1.1.0 installer had 0 detections across 70 antivirus engines.

Unsigned First Release

CometPDF 1.1.1 is not code-signed yet. Because of that, Windows SmartScreen or Chrome may show an unknown publisher warning even when antivirus engines do not flag the file as malicious.

To continue in Windows SmartScreen, click More info and then Run anyway. If you prefer to avoid installation, download the Portable ZIP instead and run CometPDF without setup.
