Convert Excel to PDF offline.

CometPDF converts XLS, XLSX, and XLSM spreadsheets to PDF. When Microsoft Excel is installed, it uses Excel's native PDF export so orientation, print areas, page breaks, and formatting stay as close as possible to printing from Excel.

This is not an online converter. CometPDF is a Windows app that converts spreadsheets offline.
Download Installer

How it works

  1. Install CometPDF for Windows.
  2. Add one or many Excel files.
  3. CometPDF creates PDFs locally using Excel export when available.

Built for real spreadsheets

Convert quotes, price lists, reports, and worksheets without uploading them to an online converter. Batch conversion and merge-to-one-PDF are included.

Fallback included

If Excel automation is unavailable for XLSX/XLSM files, CometPDF can still create a readable PDF using workbook data, sheet orientation, column widths, row heights, and basic styling.

FAQ

Does it keep Excel formatting?

When Microsoft Excel is installed, CometPDF uses Excel's own PDF export for the closest result.

Where is the PDF saved?

In the selected output folder, or next to the original file when using right-click conversion.

Can it merge spreadsheet PDFs?

Yes. Enable the merge option to create one combined PDF after conversion.
