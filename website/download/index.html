What changed in 1.1.0?

  • Added PDF to PNG and PDF to JPG export.
  • Added PDF to TXT text extraction for selectable PDF text.
  • Added PDF to Word DOCX visual export.
  • Added PDF to PowerPoint PPTX visual export.
  • Kept local no-upload conversion for Office files, images, emails, text files, and PDFs.

PDF to Word and PDF to PowerPoint in this release prioritize visual fidelity. They preserve PDF pages visually rather than rebuilding every object as fully editable content. OCR and advanced PDF page tools are planned for a future Pro release.

For email archiving, CometPDF can also convert Outlook MSG and EML email files to PDF offline, including multiple email files in one batch.

FAQ

Which download should I choose?

Choose the installer for normal everyday use, shortcuts, uninstall support, and optional right-click conversion. Choose Portable ZIP if you want to run CometPDF without installation.

Is CometPDF free?

Yes. CometPDF 1.1.0 is a free release. A future Pro version is planned for OCR, compression, watch folders, and PDF page tools.

Does CometPDF upload files?

No. CometPDF converts selected files locally on your Windows computer. It is built for private, offline, no-upload PDF conversion.

Why does Windows show an unknown publisher warning?

CometPDF is not code-signed yet, so Windows SmartScreen and Chrome may warn before running it. This is common for new unsigned Windows apps.

How can I verify the download?
