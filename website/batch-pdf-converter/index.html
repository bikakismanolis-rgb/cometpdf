Batch workflows

Real document work rarely involves one file at a time. A folder may contain proposals, spreadsheets, scanned images, emails, reports, and attachments that all need PDF copies. CometPDF is built for that kind of mixed batch: select the files, choose the conversion mode, and produce outputs without opening each source file manually.

When Microsoft Office, Outlook, or LibreOffice are installed, CometPDF can use them for print-like PDF output. That helps preserve page orientation, print areas, slide proportions, Word headers, and other formatting details while still giving users a simple batch interface.

  • Batch Word to PDF for DOC, DOCX, and RTF files.
  • Batch Excel to PDF for XLS, XLSX, and XLSM files.
  • Batch image to PDF for JPG, PNG, WEBP, BMP, and TIFF files.
  • Batch PDF to image for exporting PDF pages as PNG or JPG.

FAQ

Can CometPDF convert multiple files at once?

Yes. Add multiple supported files and run them in one batch. This is useful for folders of documents, emails, images, reports, or mixed office files.

Can it merge converted PDFs?

Yes. CometPDF can optionally merge PDF outputs into a single file after conversion. This is useful when many documents need to be delivered or archived together.

Does batch conversion upload files?

No. Batch conversion happens locally on your Windows computer. CometPDF does not send your selected files to a server.

Which file types can be batch converted?

CometPDF supports DOC, DOCX, RTF, XLS, XLSX, XLSM, PPT, PPTX, MSG, EML, JPG, PNG, WEBP, BMP, TIFF, TXT, CSV, HTML, XML, JSON, and PDF export modes.

Does it support reverse PDF batch conversion?

Yes. CometPDF 1.1 can batch export PDFs to PNG, JPG, TXT, DOCX, and PPTX. Visual export modes preserve page appearance rather than rebuilding every object as editable content.
