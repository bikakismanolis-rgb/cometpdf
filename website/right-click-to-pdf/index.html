FAQ

How do I get right-click Convert to PDF?

Use the CometPDF installer and enable the right-click integration option when available. This adds the conversion command to Windows File Explorer.

Does right-click conversion upload files?

No. Right-click conversion runs locally through CometPDF. Your selected file is not sent to a server.

Where is the PDF saved?

CometPDF saves output based on the configured output behavior. For right-click conversion, it is designed to keep the result easy to find near the original file or in the selected output folder.

Does the portable ZIP support right-click conversion?

The portable version is meant to run without setup, so it does not provide the same Windows Explorer integration as the installer.

Which files can I right-click convert?

CometPDF supports common Office files, images, emails, text and data files, and PDFs depending on the selected conversion mode.
