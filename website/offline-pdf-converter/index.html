FAQ

Does CometPDF work without internet?

Yes. After installation, CometPDF converts files entirely offline. No internet connection is needed for normal file conversion.

What file types does CometPDF support?

CometPDF supports DOC, DOCX, RTF, XLS, XLSX, XLSM, PPT, PPTX, MSG, EML, JPG, PNG, WEBP, BMP, TIFF, TXT, CSV, HTML, XML, JSON, and existing PDF files. It also includes reverse PDF export tools for PNG, JPG, TXT, DOCX, and PPTX.

Is there a file size limit?

There is no server-side limit because files are not uploaded. File size is limited by your local machine's memory and the application being used for conversion, such as Word or Excel.

Does it work on Windows 10 and Windows 11?

Yes. CometPDF runs on Windows 10 and Windows 11. It is built as a desktop application for common office and document workflows.

Is CometPDF free?

Version 1.1.1 is free. A Pro version with OCR, compression, watch folders, and PDF page tools is on the roadmap.
