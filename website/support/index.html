CometPDF support and feedback.

CometPDF is a young free Windows PDF converter, and user feedback is the best way to improve it. Report bugs, request file format support, suggest workflow improvements, or ask for help by email or through GitHub Issues.

For direct support, email support@cometpdf.com. Public bug reports and feature requests can also be tracked through GitHub Issues.

What to report

If a file does not convert correctly, the most useful report includes the file type, Windows version, whether Microsoft Office or LibreOffice is installed, and what happened during conversion. You do not need to upload private files. A clear description or a sample non-confidential file is enough to start.

  • Conversion errors
  • Layout problems
  • Missing file type support
  • Right-click conversion issues
  • Installer or Portable ZIP problems
  • Feature requests for future versions

Feature requests

Planned future ideas include OCR, PDF compression, watch folders, more PDF page tools, and stronger Microsoft Store distribution. GitHub Issues is the right place to vote for priorities and describe real workflows that CometPDF should support.

Useful requests explain the problem, the file type, and the expected result. For example: "I need to convert scanned invoices to searchable PDF" is more useful than "add OCR" because it explains the workflow behind the feature.

FAQ

Where should I report a bug?

Use GitHub Issues for public bug reports, or email support@cometpdf.com for direct support. Include the file type, conversion mode, Windows version, and the error message if one appeared.

Do I need to upload my private file?

No. Do not upload private documents. Describe the problem or use a non-confidential sample file if needed.

Can I request a feature?

Yes. GitHub Issues can be used for feature requests, supported format ideas, and workflow suggestions.

Is CometPDF open source?

The project is hosted on GitHub and public release notes are available there. Use the repository for current project files, releases, and issue tracking.

Where can I download the latest version?

The recommended download page is cometpdf.com/download. GitHub releases are also available for users who prefer GitHub.
