# CometPDF Product Plan

## Free Edition

- Convert common files to PDF.
- Office native export when Microsoft Office is installed.
- Image, text, email, and PDF support.
- Batch conversion.
- Merge all outputs into one PDF.
- Right-click conversion on Windows.

## Pro Ideas

- OCR for scanned PDFs and images.
- PDF compression.
- Watch folder automation.
- Split/reorder/delete PDF pages.
- Watermark and stamp presets.
- Company branding.
- Priority support.
- Web/mobile upload interface backed by a Windows conversion machine.

## Distribution Checklist

- Add a real app icon.
- Add a small end-user license or open-source license.
- Add installer.
- Add code signing if selling commercially.
- Test on a clean Windows machine with and without Microsoft Office.
